# Run4Dart Core

This repository is used to learn and practice core programming concepts,
Git & GitHub workflow, Python fundamentals, data structures, and AI basics.

## Repository Structure

- experiments/ → Small experiments and learning tests
- docs/ → Notes, explanations, and references
- app/ → Actual application code (added later)

## Goal

Build strong foundations before developing large-scale applications
and preparing for internships and open-source programs like GSoC.
